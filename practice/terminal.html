What's Vim

Vim is a highly configurable, keyboard-driven, free, and open-source text editor

Now We Can Begin

First off we need to figure out what to Vim, for the sake of the guide, we will create a document that we will use.

  1. Go to terminal
  2. Type: cd Desktop, then click enter
  3. Type: touchindex.htm, then click enter
  4. Now type in: vimindex.html and a big text box will appear

Now you know how to get in vim, here's how to use it.

  • To move around Vim, use arrow keys
  • To edit the text, press I
  • To exit editing mode, press esc
  • To save and exit, type :wq while being in viewing mode.
