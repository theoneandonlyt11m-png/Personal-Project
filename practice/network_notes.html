Definitions

         Protocol - set of rules that regulate how computers sent and recieve data.
         
         Packet -  a small, formatted chunk of data sent over a network, 
         containing both the actual data and crucial control information.
         like the source and destination addresses.
         
         Header - the control information at the beginning 
         of a data packet, containing metadata like the source 
         and destination IP addresses, packet length, and version.
         
         Payload -  the actual data being sent over a network.
         
         Router - a networking device that connects local networks 
         to each other and to the internet, directing data 
         packets between them to ensure they reach the correct destination.
         
         Routing Table - a routing table is a network device's internal address book or map.
